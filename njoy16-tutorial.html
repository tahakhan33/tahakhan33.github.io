Compiling NJOY2016 on Linux

Updated: July 2026 10 Min Read Nuclear Data Processing

This guide covers the necessary steps to build the NJOY2016 nuclear data processing system from source, ensuring you have the proper Fortran compilers installed to handle cross-section generation.

Step 1: Install Dependencies

NJOY heavily relies on Fortran. You need to install gfortran alongside the standard build utilities:

sudo apt update
sudo apt install git cmake build-essential gfortran

Step 2: Clone and Build

Pull the latest stable release from the official repository and use CMake to compile the executable.

git clone https://github.com/njoy/NJOY2016.git
cd NJOY2016
mkdir build
cd build
cmake ..
make
sudo make install

Module Pro-Tip: When writing your inputs for the MODER module to convert ENDF tapes, ensure the output tape number is set to a positive 20 (not -20) to strictly align with formatting standards.

Step 3: Verification

Test the installation by checking the executable path or running one of the built-in test suites located in the repository.
